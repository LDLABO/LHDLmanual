引用

引用は下のように書きます、引数は入力も出力も複数持てます。 

logicname sample

entity add
input  a[4];   {引数1}
input  b[4];   {引数2}
output c[4];   {引数3}

   c=a+b;
ende

entity main
input  a[4],b[4];
output c[4];

   part add(a, b, c)
      { ↑  ↑ ↑ ↑}
      { 引  引 引 引}
      { 用  数 数 数}
      { 名  1  2  3 }
ende
endlogic
