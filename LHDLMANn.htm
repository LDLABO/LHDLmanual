定数

定数は10進数、16進数、2進数が使えます。指定がなければ10進数、頭に0bと付けると2進数、0x と付けると16進数になります。 

logicname sample

entity main
input  a[8];
output b[8];
bitn   c[8];   {←非記憶}

   c=5;   {cは定数と同じ}

              {2進数}
               {↓}
   b = a + c + 0b101 + 0xAa;
          {↑}         {↑}
          {↑}        {16進数}
  {a と同じ桁数が必要}

ende
endlogic

上の例では c は非記憶の内部信号で割り当てられていますが、定数を代入しただけの非記憶の内部信号は信号の実体は論理式からは無くなり定数と同じになります。
結果は定数と同じになりますがコンパイルの過程では信号ですので演算子を使った加算では a と c は同じ桁数でないと正しい論理になりませんので、その点は注意してください。
